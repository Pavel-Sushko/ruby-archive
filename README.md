# Ruby Archive

## Description

This is a simple Ruby script that moves expired files to a backup directory.

## Usage

```bash
ruby archive.rb -s <source> -d <destination> [-t <time>]
```

### Options

| Option | Description |
| ------ | ----------- |
| -s     | Source directory |
| -d     | Destination directory |
| -t     | Time in days (default: 30) |
